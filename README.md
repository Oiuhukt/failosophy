# failosophy

My personal website built with Jekyll and the Tufte-Jekyll theme.

## About

This site hosts teaching materials, philosophical drafts, references, and other academic content. It is designed with a focus on **readability**, **accessibility**, and **clean typography**.

## Features

- Two-color scheme (high contrast for accessibility)
- Spanish language support with hyphenation
- Responsive design with fluid typography
- Sidenotes, margin notes, and epigraphs
- WCAG 2.1 AA compliant
- Montserrat font for optimal legibility
